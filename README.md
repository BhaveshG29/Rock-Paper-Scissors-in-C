# Rock-Paper-Scissors-in-C
Rock Paper Scissors

A game which scans integers. 
1 for Rock
2 for Paper
3 for Scissors

A randomizer for the Computer's move using the function rand_move().

check_win() functions which checks if the player has won or Loss.

switch implementation, which allows to print player and computer's move.


Requirements:
Compliler's like gcc or clang, or any other C compiler.
